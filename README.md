# Email Automation with Make + OpenAI 📧

Automated personalized email system built with Make.com and OpenAI.

## How it works

Google Sheets → OpenAI → Google Sheets (log) → Gmail

1. Reads client data from Google Sheets (name, product, status)
2. Generates personalized email via OpenAI GPT-4o-mini
3. Saves generated text back to Google Sheets
4. Sends email to client via Gmail

## Features

- Personalized emails based on client status
- Grammar check built into the prompt
- Automatic logging to Google Sheets
- Works for any number of clients

## Tech Stack

- Make.com (no-code automation)
- OpenAI GPT-4o-mini
- Google Sheets
- Gmail
